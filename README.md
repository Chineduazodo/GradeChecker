# Grade Checker Console Application

A beginner-friendly C# console application that calculates and displays letter grades based on student scores. This project was built to practice core C# concepts such as variables, conditional statements, loops, and exception handling.

## Features

* Accepts student information and scores
* Calculates grades based on score ranges
* Handles invalid input
* Allows multiple grade checks in a single session

## Technologies Used

* C#
* .NET

## Concepts Practiced

* Variables
* User Input and Output
* If-Else Statements
* Do-While Loops
* Try-Catch Exception Handling

## What I Learned

This project helped me understand program flow, input validation, debugging, and how different C# concepts work together to build a functional application.
